# Financial-Transaction-Dataset
A financial transaction dataset contains structured records of monetary activities, including transaction ID, date, amount, sender, receiver, and payment method. It is widely used for financial analysis, fraud detection, auditing, and understanding customer behavior across banking and digital payment systems.
